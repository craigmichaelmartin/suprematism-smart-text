Explanation of Features

Rows: Specify a maximum number of rows for the display text, using an ellipsis with popover if text overflows.
Editable: Specify the display text to be editable.
Placeholder: Specify a placeholder in editable area if empty.
Default: Specify a default value if the editable area is saved empty (or starts empty).
Forced: Specify the editable area cannot be saved/cancelled until a value is provided (if starts empty, component will begin as editable).
